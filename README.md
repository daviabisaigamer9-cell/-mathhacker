# 🧠 MathHacker — Plataforma SaaS Educacional

O **MathHacker** é um ecossistema completo e tático projetado para potencializar o desempenho de alunos de matemática da Rede Elite (Capítulos 5, 6 e 8 do OED: Monômios, Polinômios, Binômios e Trinômios). 

Alimentado por Inteligência Artificial de ponta e infraestrutura de segurança avançada.

## 🚀 Recursos do Sistema

- **🤖 Mascote 'Sexta-Feira' (Gemini 2.5 Flash)**: Assistente inteligente capaz de resolver equações passo a passo, ler imagens de exercícios e contextualizar explicações com gírias amigáveis.
- **🎮 Gamificação por Nível e XP**: Aumento gradual de ranking com base em interações.
- **🛡️ Acesso Premium via Paywall**: Controle via chaves únicas de acesso.
- **📊 Dashboard do Fundador**: Acompanhamento de queries em tempo real.

## 🛠️ Stack Tecnológica

- **Backend**: Node.js, Express.js, SQLite3
- **Frontend**: HTML5, Tailwind CSS, JavaScript (ES6)
- **IA**: Google Generative AI SDK

## 💾 Como Rodar Localmente

1. Instale as dependências:
```bash
npm install
```

2. Defina sua chave da API do Gemini:
```bash
set GEMINI_API_KEY=sua_chave_aqui
```

3. Inicie o ambiente:
```bash
npm start
```
